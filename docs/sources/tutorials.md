# Business Links tutorials

The **Business Links panel** for Grafana offers a seamless way to navigate using external links, internal dashboards, and dynamic dropdown menus. This section provides detailed guides, key features, and practical use cases.

This page contains step-by-step video tutorials that showcase new features and address common challenges. These tutorials help you get the most out of Business Suite.

## Unlock enhanced interactivity with Business Links

The Business Links panel overcomes the limitations of Grafana's native links, offering greater flexibility and interactivity for your dashboards. For insights and examples, refer to the [blog post](https://volkovlabs.io/blog/business-links-power/).
